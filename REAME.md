# WatchCorona

Compact information about the current coronavirus pandemic.

## Getting Started

### Installing

Download this repository and install the necessary node modules
```
git clone https://github.com/radiachk/WatchCorona.git
cd WatchCorona
npm install
```

### Running

Start the expo project 
```
expo start
```
